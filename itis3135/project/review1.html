Design:

  • Contrast/font sizing: The contrast and font sizing seem sufficient for readability

  • Site colors/fonts using standard .css file: A brand logo is visible, I think it would be cool if it also appeared as the icon for the website. The logo is an odd shade of gray that isn’t seen in the other parts of the page the font in the logo is never seen elsewhere.

CRAP:

  • Contrast: The contrast between text and background is good, making it easy to read.

  • Repetition: Repetition of design elements seems minimal but consistent.

  • Alignment: Elements appear to be aligned properly only thing is the images used as the icons in the “contact” section have more spacing visually than any other images on the page.

  • Proximity: Related elements are appropriately grouped together.

Page Structure:

  • Header/Main/Footer: The page structure includes a header, main, and footer.

  • Header: The header includes the site/brand header in an h1 tag. The page name is not included in the header.

  • Main: The main section starts with the name of the page as an h2.

  • Footer: The footer includes a menu for the user's pages but the style of the footer doesn’t match that of the header. The header has an outline that separates it from the content of the page but the footer doesn’t have an outline which makes it seem unlike the rest. Also I'm unsure why you have your certifications in the footer because this website is for the client so why does it have your information? I think all that is needed is that the website was created by your firm.

Other Requirements:

  • Brand tagline: Branding is visible in the landing page withe the name centered and in big lettering to distinguish itself from the restof the page.

  • HTML/CSS validation: Buttons/links for HTML and CSS validation are present on the page and they also work.
