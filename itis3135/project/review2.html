CRAP:

  • Contrast: The contrast between text and background is adequate, ensuring readability. However, font sizing could be improved in skills sections to enhance readability, particularly for smaller screens or users with visual impairments. I see this in the "skills" section.

  • Repetition: Design elements are consistent throughout the site, which is good for creating a cohesive user experience. However, there could be more repetition of certain visual elements to strengthen the site's branding and identity with the styling of the iconography being in other places besides just the header.

  • Alignment: Elements are aligned properly, contributing to a clean and organized layout. However, there are a few instances where alignment could be adjusted slightly for better visual balance specifically in the “skills” section because it’s all horizontally placed with no spacing on the sides. The resume in that same section is also too small to read so laying them out vertically would be preferred.

  • Proximity: Related elements are grouped together logically, making it easy for users to understand the relationships between different pieces of content. However, in the bullets for the "skills" sections, tighter proximity between related elements could further improve clarity and organization.

Page Structure:

  • Header/Main/Footer: The page structure includes a header, main content area, and footer, which is a standard and effective layout for websites.

  • Header: The header contains the site/brand name in an h1 tag, which is good for establishing the site's identity. However, it could benefit from additional elements such as a navigation menu or brand slogan to enhance usability and branding. Also there is a lack of spacing in the header with text almost being outside the navigation bar.

  • Main: The main content area starts with the page title as an h2, which helps users understand the context of the content. However, providing more descriptive headings for subsections within the main content could improve navigation and comprehension.

  • Footer: The footer includes a menu for the user's pages, which is helpful for site navigation. It’s simple but consistent.

Other Requirements:

  • Brand tagline: I am not sure how the logo correlates to Melissa because I don’t see that connection anywhere else. I like that you have it as the website icon though

  • HTML/CSS validation: Buttons/links for HTML and CSS validation are present on the page and they also validate properly.
